Exploratory Data Analysis

  1. Data (Resale Flat Prices) is downloaded directly from http://data.gov.sg.
  2. The public housing resale transacted prices are provided by Housing Development Board in Singapore.
  3. This analysis uses all available data in the downloaded data size. The towns are further classified into regions based on information found on this website here.
  4. A more interactive application (ShinyApps) developed to explore the data can be found here.
  5. All the charts provided in this presentation are generated using R ggplot embedded in this slidify document.
